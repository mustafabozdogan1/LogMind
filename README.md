# 🚀 LogMind

LogMind is a Python-based log analysis and visualization tool designed to process, analyze, and visualize log data efficiently.

---

## 🔍 Features

- Log parsing from text files
- Detection of ERROR, WARNING, INFO messages
- Top error frequency analysis
- JSON report generation
- Data visualization with matplotlib
- Timeline-based log event analysis

---

## 📊 Example Output

ERROR: 2  
WARNING: 1  
INFO: 2  

Top Errors:
1. Database connection failed
2. API timeout

---

## 📈 Generated Charts

- Error Distribution Chart
- Top Errors Chart
- Timeline Chart

All charts are automatically saved as `.png` files.
